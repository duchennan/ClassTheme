CSS stands for cascading style sheet。用来设计一个网页的外观，他是众多语言之一。
HTML is concerned with the content and structure of a website。跟内容有关，跟结构有关。

HTML页面的组织结构
HTML有一个基本的结构，这个基本结构就是我们要掌握的。
从HTLM5语言以后，第一条语言就是这个document type。说明这个语言到底是哪个版本。
第二个重要的事情，HTML就整个结构的最大的根。为什么叫它最大的根呢？因为整个HTML语言就像一个树根，在这之下有两个巨大的树杈，第一个树杈就是这个head，头标记，或者我们把它叫头元素。第二个重要的就是body身体。我们作为初学者写的所有东西，都在body里面。
HTML语言或者HTML文件的一个基本结构就这么点。

浏览器和Web
谈一谈web浏览器，它的工作原理。首先，站在文件的层面来理解写的每一张网页，它由代码构成的，代码就是内容，对吧，然后这内容一存盘，它一定会形成一个文件，这个文件它放在电脑上。它也可以跟你电脑上的文件发生链接的关系，如果它放在网络的服务器上，可以跟全世界的服务器的文件也发生了关系。
放在你电脑本地，它也可以跟互联网发生的关系。在HTML这个语言当中的几个简单的标签，第一个img插图，插图的src所指向的是什么？
就是一个图片的什么地址，图片在哪里，图片就在你电脑本地。
引用的图片，视频、音频也是文件，当你用HTML写完了以后，它们都被浏览器抓取了，然后一张网页就出来了。

代码编辑器
谈谈代码编辑器，记事本的效率是非常低的，如果追求高效率，那就要代码编辑器了。
他可以干什么？第一件事情，开启文件和存盘，再也不要去选文件后缀名了。第二件事情，也是最重要的，代码高亮--关键字有关键字的颜色，属性有属性的颜色，文本有文本的颜色，注释有注释的颜色。
还有就是能够重点的表达结构，可以折叠代码。
