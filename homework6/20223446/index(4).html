CSS盒子模型

CSS 框模型是 CSS 的一个模块，用于定义矩形框（包括其填充和边距），这些矩形框是为元素生成并根据可视格式模型进行布局的。


箱体模型概述
CSS 中的框由内容区域组成，内容区域显示任何文本、图像或其他 HTML 元素。可以选择在一条或多条边上用填充、边框和边距包围。盒子模型描述了这些元素如何协同工作以创建由 CSS 显示的盒子。要了解有关它的更多信息，请阅读 CSS 盒模型简介。

框边关键字
盒子模型规范定义了一组关键字，这些关键字引用盒子每个部分的边缘，这些关键字在 CSS 中用作关键字值，包括作为盒子大小属性的值，以控制盒子模型如何计算其大小。
